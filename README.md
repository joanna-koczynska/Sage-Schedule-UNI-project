The application named Sage Schedule is an interactive planner combining the Bullet Journal organization method with the Pomodoro time management technique. Its goal is not only task planning but also supporting productivity, 
managing user energy, and taking care of mental health.

Key Features:
- Calendar and Planner: Enables planning in monthly, weekly, and daily views (dailies). The user can add entries in the following categories: task, event, and important.
- Pomodoro Module: A built-in timer measuring 25-minute work sessions and breaks, enriched with 3D animations (e.g., a burning candle) and a Lo-Fi music widget (Spotify) for better focus.
- Habit Tracker: A tool for tracking and checking off weekly habits.
- Daily Mental Health: A section for well-being control, covering mood monitoring, sleep duration, body hydration, and a gratitude journal.
- Bookshelf: A virtual catalog of read or planned books with the ability to rate them and add notes.
- Extras: A quick notepad and a weather widget displaying the temperature in a selected city.

Technology and Design: 
  The application is a web solution (running in a browser), built based on the Node.js environment and an SQLite database.
